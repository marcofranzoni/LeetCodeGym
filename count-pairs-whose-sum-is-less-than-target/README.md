# 2824. Count Pairs Whose Sum is Less than Target

[url](https://leetcode.com/problems/count-pairs-whose-sum-is-less-than-target/description/)

Given a 0-indexed integer array nums of length n and an integer target, return the number of pairs (i, j) where `0 <= i <
j < n` and `nums[i] + nums[j] < target`.

## Constraints:

1. 1 <= nums.length == n <= 50
2. -50 <= nums[i], target <= 50
